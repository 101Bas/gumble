# gumble

## Sub-projects

- gumble
    - Client library
- gumble_openal
    - [OpenAL](http://kcat.strangesoft.net/openal.html) audio system for gumble
- gumble_ffmpeg
    - [ffmpeg](https://www.ffmpeg.org/) audio source for gumble
- gumbleutil
    - Extras that can make working with gumble easier
