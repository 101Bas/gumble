# gumble

## Sub-projects

- gumble
    - Client library
- gumble_openal
    - [OpenAL](http://kcat.strangesoft.net/openal.html) audio system for gumble
- gumbleutil
    - Extras that can make working with gumble easier
- barnard
    - Terminal-based client which uses the gumble library
